# Slack Plugin for Conductor

Install: `conductor install slack`

## Setup

**Authentication:** Bot Token

```bash
conductor slack setup
`conductor plugins enable slack`
```

Get credentials at: https://api.slack.com

## Tools

```
slack_send_message, slack_channels, slack_read_channel, slack_search, slack_users, slack_add_reaction
```

# Google Calendar Plugin for Conductor

Install: `conductor install gcal`

## Setup

**Authentication:** Google OAuth

```bash
conductor auth google
`conductor plugins enable gcal`
```

Get credentials at: https://developers.google.com/calendar

## Tools

```
gcal_list, gcal_get, gcal_create, gcal_update, gcal_delete, gcal_list_calendars
```

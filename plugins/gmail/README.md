# Gmail Plugin for Conductor

Install: `conductor install gmail`

## Setup

**Authentication:** Google OAuth

```bash
conductor auth google
`conductor plugins enable gmail`
```

Get credentials at: https://developers.google.com/gmail/api

## Tools

```
gmail_list, gmail_read, gmail_search, gmail_send, gmail_reply, gmail_mark_read, gmail_trash
```
